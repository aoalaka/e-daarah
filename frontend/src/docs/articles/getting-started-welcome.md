# Welcome to E-Daarah

E-Daarah is a school management platform built specifically for **madrasahs and Islamic schools**. It replaces the spreadsheets, WhatsApp groups, and paper registers most schools rely on with one place to track attendance, fees, Qur'an progress, courses, and parent communication.

## Who uses E-Daarah?

- **Admins** — set up the school, manage classes and students, send fee reminders, view reports.
- **Teachers** — record daily attendance, mark Qur'an sessions, enter exam grades, track course progress.
- **Solo teachers** — single-teacher madrasahs that want all the tools without a separate admin.
- **Parents** — open the parent portal to see their child's attendance, grades, fees, and Qur'an position any time.

## What you can do with it

- **Track attendance** — daily roll call with optional dressing, behaviour, and punctuality grades.
- **Record exams** — per subject, per semester, with class averages and rankings.
- **Track Qur'an progress** — Hifdh, Tilawah, Muraja'ah; with surah/ayah position tracking per student.
- **Manage courses** — create curriculum units and track who's taught what (per-student or whole-class mode).
- **Manage fees** — schedule, record payments, send reminders, generate per-family statements.
- **SMS parents** — manual sends, fee reminders, attendance alerts. Multi-child families get one message per phone.
- **Reports** — student reports, class performance, rankings, fee statements, course coverage.
